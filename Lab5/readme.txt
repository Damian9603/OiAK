Program przyspieszający działanie programu całkującego. Program przyspiesza działanie funkcji około trzykrotnie. Wyniki całek i stosunek szybkości jest wyświetlany po uruchomieniu programu. Zawartość katalogu:
main.c - właściwy program w C
my_cpuid.s - funkcja mierząca czas
sse.s - funkcja licząca całkę za pomocą poleceń SSE
x.s - funkcja licząca całkę za pomocą poleceń FPU
